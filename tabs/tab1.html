Проект 1: Космическая стрелялка

В активной разработке Старт: Январь 2024

О проекте

Это игровой проект, разрабатываемый на движке Unity. Игра представляет собой аркадный шутер с процедурной генерацией уровней, системой улучшений корабля и динамическим ИИ противников.

Процедурная генерация

Каждый уровень уникален благодаря продвинутой системе генерации

Умные противники

ИИ адаптируется к стилю игры пользователя

Система прогрессии

Множество улучшений и кастомизаций корабля

Используемые технологии

  • Unity Engine 2022.3+
  • C# с использованием DOTS
  • Blender для 3D моделей
  • Git для контроля версий
  • Custom netcode для мультиплеера
  • Shader Graph для визуальных эффектов
