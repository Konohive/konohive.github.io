Как работает этот форум?

На базе GitHub Issues

Каждая тема форума — это Issue в репозитории Konohive/konohive.github.io. Все обсуждения хранятся непосредственно на GitHub.

Авторизация через GitHub

Для создания тем и комментариев нужен аккаунт GitHub. Нажмите на тему → перейдёте на GitHub → авторизуйтесь → пишите комментарий.

Безопасность и надёжность

Ваши данные в безопасности. Форум будет работать до тех пор, пока работает GitHub. Даже если этот сайт перестанет работать, все обсуждения останутся в вашем репозитории.

Автоматическое обновление

Темы обновляются в реальном времени через GitHub API. Новые комментарии появляются автоматически без перезагрузки страницы.

Технические детали:

  • Репозиторий: Konohive/konohive.github.io
  • API: GitHub REST API v3
  • Обновление: Каждые 60 секунд
  • Хранение данных: Issues вашего репозитория
